2. WHAT DEXTER IS (for the application)

One-liner: Dexter is the agent economy platform – AI agents discover tools via MCP, pay via x402 micropayments on Solana, and get results. Same brain, same wallet, infinite interfaces.

The ecosystem:

  • x402 Protocol: HTTP-native micropayments on Solana (plus Base, Polygon, Avalanche, Arbitrum, Optimism). Open SDK: npm install @dexterai/x402. Any developer can monetize any API in three lines of code.
  • MCP Marketplace: 50+ tools across 14 toolsets – trading (Jupiter swaps, Hyperliquid perps), research (web search, on-chain analytics), media (Sora video, meme generation), social (Twitter sentiment). All payable via x402.
  • Facilitator: Settlement infrastructure at x402.dexter.cash. Verifies payments, signs settlements, sponsors gas fees. Multi-chain. Live.
  • Multi-Platform Connectors: Works natively in ChatGPT, Claude, Cursor, Telegram, Alexa, X (@dexteraiagent), and any MCP client. OAuth flow connects it all.
  • Voice Interface: WebRTC realtime voice at beta.dexter.cash. Talk to the agent, it executes trades, runs research, generates media.
  • ERC-8004 Trust Layer + A2A Agent Cards: On-chain identity and portable reputation. Agents discover each other autonomously.
  • Dexter Lab: AI-powered builder at lab.dexter.cash. Anyone can build, deploy, and monetize a paid API endpoint from a browser – describe it, AI builds it, one-click deploy, live in minutes.
  • PokéDexter: Wagered Pokémon battles on Solana ($1-$25 USDC, trustless escrow).
  • Clawdexter: Give Claude agents autonomous x402 payment capability.
  • $DEXTER Token: Revenue from the platform buys back $DEXTER. Token-gated access tiers.

What makes it real:
